이 가이드에 대하여

이 페이지는 선샤인 언더그라운드 내의 Ruby on Rails 프로젝트 작업시 원활한 협업을 돕기 위한 html/css 가이드 입니다. SCSS 사용법 에 대한 사전 지식이 필요합니다.

코딩 스타일

간격

  • 각 스타일 요소들끼리는 줄넘김을 합니다.
  • 블록을 닫을때는 언제나 새로운 줄에서 닫아 주세요.

포맷

  • 색 지정은 rgba()를 사용하는 경우가 아니면, #000와 같은 hex 코드로 지정해주세요.
  • 0의 값을 지정할때는 단위를 넣지 않습니다. 예: margin: 0;
  • 단축 지정은, 모든 값을 지정해야할때 이외에는 사용하지 않습니다.

기타

  • SCSS 에서 3단 이상의 Nesting 은 사용하지 않도록 합니다.
  • KSS로 스타일을 문서화 합니다.(성완)

예시


// 좋은 포맷 예시
.styleguide-format {
  color: #000;
  background-color: rgba(0, 0, 0, .5);
  border: 1px solid #0f0;
}

// 단축 지정은, 자제해주세요.
.not-so-good {
  margin: 0 0 20px;
}
.good {
  margin-bottom: 20px;
}

요소의 특수성

페이지에서 한번만 나오는 요소는 ID를 사용해야 합니다, 그 외에는 클래스를 사용하십시오.

  • id를 사용하기 좋은 것들: 헤더, 푸터, 모달 팝업
  • id를 사용하기 나쁜 것들: 헤더, 푸터, 모달 팝업
